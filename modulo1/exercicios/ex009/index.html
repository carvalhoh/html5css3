Listas Ordenadas

Para criar listas ordenadas, é usada abre tag <ol> e fecha tag </ol>, e dentro dela para para inserir os itens, é usada abre tag <li>, com o HTML5 não é obrigatório fechar a tag </li>.
Também é possível personalizar a lista, usando o parametro type="", e dentro das aspas 1, A, a, I, i.

Lista ordenada com número:

        <ol type="1">
        <li>Acordar</li>
        <li>Escovar os dentes</li>
        <li>Tomar café da  manhã</li>
        </ol>
Como Fica:
  1. acordar
  2. escovar os dentes
  3. Tomar café da manhã

Listas não ordenadas

Para criar listas não ordenadas, é usada abre tag <ul> e fecha tag </ul>, e dentro dela para para inserir os itens, é usada abre tag <li>, com o HTML5 não é obrigatório fechar a tag </li>.
Também é possível personalizar a lista, usando o parametro type="", e dentro das aspas circle, square, disc.

Lista não ordenada com square:

    <ul type="square">
    <li>Pão</li>
    <li>Leite</li>
    </ul>
Como Fica:
  • Pão
  • Leite

  • Leite
